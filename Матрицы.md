## Поворот матрицы против часовой стрелки

^af72b0

```Python
m = [
	 [1, 2, 3],
	 [4, 5, 6],
	 [7, 8, 9],
]
```

```Python
list(zip(*m))[::-1]

'''
[
	(3, 6, 9), 
	(2, 5, 8), 
	(1, 4, 7)
]
'''
```
 
 > После поворота вложенные списки будут преобразованы в кортежи, ввиду особенности функции `zip`.
